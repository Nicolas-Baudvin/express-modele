# Modèle / Template pour nodejs - express - mongoDB

## Installation

```bash
git clone https://github.com/Nicolas-Baudvin/express-modele.git
```

```bash
cd express-modele
```

```bash
npm install
```

```bash
npm link
```

Accès global à la commande "express-modele" pour utiliser ce template dans un autre projet :

```bash
cd mon_projet
```

```bash
express-modele
```
